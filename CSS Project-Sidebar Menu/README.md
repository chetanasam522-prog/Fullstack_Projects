
## Photography Sidebar Menu Project

- A simple and responsive Sidebar Navigation Menu built using HTML and CSS only (No JavaScript).
- This project uses the CSS Checkbox Hack to toggle the sidebar smoothly.

-----

## Features

-  Sidebar Navigation Menu
-  Background Image Layout
-  Smooth Slide Animation
-  Hover Effects
-  Font Awesome Icons
-  Google Fonts (Poppins)
-  Pure HTML & CSS (No JavaScript)

-----

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome Icons
